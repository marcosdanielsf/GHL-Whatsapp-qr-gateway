## PASSO 2: CRIAR INSTANCIA (3 min)

### 2.1 No Painel Z-API
```
1. Clique em "Nova Instancia"
2. Escolha um nome (ex: "mottivme-calls")
3. Clique em "Criar"
```

### 2.2 Conectar WhatsApp
```
1. Clique na instancia criada
2. Vai aparecer um QR Code
3. No seu celular: WhatsApp > Aparelhos Conectados > Conectar
4. Escaneie o QR Code
5. Aguarde conectar (10-30 segundos)
```

### 2.3 Verificar Status
```
Status deve mostrar: "Conectado" (verde)
```

## PASSO 4: TESTAR CHAMADA (5 min)

### 4.1 Via Postman/Insomnia

```http
POST https://api.z-api.io/instances/{INSTANCE_ID}/token/{TOKEN}/send-call

Headers:
  Client-Token: {SEU_CLIENT_TOKEN}
  Content-Type: application/json

Body:
{
  "phone": "5511999999999",
  "callDuration": 10
}
```

### 4.2 Via cURL

```bash
# Substitua as variaveis
INSTANCE="seu-instance-id"
TOKEN="seu-token"
CLIENT_TOKEN="seu-client-token"
TELEFONE="5511999999999"

curl -X POST "https://api.z-api.io/instances/$INSTANCE/token/$TOKEN/send-call" \
  -H "Client-Token: $CLIENT_TOKEN" \
  -H "Content-Type: application/json" \
  -d "{\"phone\": \"$TELEFONE\", \"callDuration\": 10}"
```

### 4.3 Response Esperado

```json
{
  "zaapId": "3999984263738042930CD6ECDE9VDWSA",
  "messageId": "D241XXXX732339502B68",
  "id": "D241XXXX732339502B68"
}
```

**E o telefone destino vai TOCAR!** 🔔

## PASSO 6: CONFIGURAR WEBHOOK (opcional)

### 6.1 No Painel Z-API

```
Instancia > Configuracoes > Webhooks

URL: https://seu-servidor.com/api/zapi/webhook
Eventos: Todos (ou selecione especificos)
```

### 6.2 Eventos Importantes

| Evento | Quando dispara |
|--------|----------------|
| `ReceivedCallback` | Mensagem recebida |
| `MessageStatusCallback` | Status da msg (enviada/entregue/lida) |
| `OnCallReceived` | Chamada recebida |
| `OnCallRejected` | Chamada rejeitada |

### 6.3 Payload de Chamada Recebida

```json
{
  "event": "OnCallReceived",
  "phone": "5511999999999",
  "isVideo": false,
  "callId": "xxx",
  "timestamp": 1706810400
}
```

## INTEGRACAO COM SOCIALFY NEXUS

### Adicionar ao .env

```bash
# Z-API Config
ZAPI_ENABLED=true
ZAPI_INSTANCE_ID=seu-instance-id
ZAPI_TOKEN=seu-token
ZAPI_CLIENT_TOKEN=seu-client-token
ZAPI_CALL_DURATION=10
```

### Usar o servico (ja documentado em 07.)

```typescript
import { ZApiService } from './services/zapi.service';

const zapi = new ZApiService({
  instanceId: process.env.ZAPI_INSTANCE_ID,
  token: process.env.ZAPI_TOKEN,
  clientToken: process.env.ZAPI_CLIENT_TOKEN
});

// Fazer chamada
await zapi.makeCall({
  phone: '5511999999999',
  callDuration: 10
});
```

## TROUBLESHOOTING

### Erro: "Instance not connected"
```
Causa: WhatsApp desconectou
Solucao: Escanear QR novamente no painel
```

### Erro: "Invalid phone number"
```
Causa: Formato errado do numero
Solucao: Usar formato 5511999999999 (sem +, sem espacos)
```

### Erro: "Client-Token invalid"
```
Causa: Token errado ou faltando
Solucao: Verificar header Client-Token (nao confundir com Token da URL)
```

### Chamada nao completa (toca e para)
```
Causa: callDuration muito curto
Solucao: Aumentar para 10-15 segundos
```
